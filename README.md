# UI-автотесты страницы регистрации Ростелеком ID

## Описание проекта
Данный репозиторий содержит набор UI-автотестов для страницы регистрации
пользователей Ростелеком ID.

Автотесты проверяют клиентскую валидацию формы регистрации в соответствии
с требованиями:
- Поле «Имя»
- Поле «Пароль»
- Поле «Подтверждение пароля»

Тесты реализованы с использованием Playwright и PyTest и имитируют
действия реального пользователя в браузере.

## Что проверяют тесты

### Поле «Имя»
- Минимальная длина (2 символа)
- Использование кириллицы
- Возможность ввода имени с тире
- Обязательность заполнения поля

### Поле «Пароль»
- Минимальная длина 8 символов
- Наличие заглавной буквы
- Использование только латинских букв
- Совпадение пароля и подтверждения
- Корректный пароль без ошибок

## Требования
- Python 3.8+
- PyTest
- Playwright
